Example 2.2

Lab 999 typed three samples for center 789 for the ABDR project.
The allele-level typing was performed on Dec 8, 2003.
They performed a series of SSO tests.

Rather than reporting a list of the tests used, the lab wants to report only a reference to a predetermined list of tests.

Note that the test scores are different than the previous example. By using the reference to a list of tests, the order of the scores must match the order of the tests in the predetermined list.
For each test performed on a sample, a score was given to indicate success (8) or failure (1).
The results are: 

    NMDP ID: 8888-6666-1    DRB1*01RY, DRB1*13GPE
    NMDP ID: 8888-6666-2    DRB1*03WKH, DRB1*04AFMT
    NMDP ID: 8888-6666-3    DRB1*04AFMK, DRB1*04AGDC
